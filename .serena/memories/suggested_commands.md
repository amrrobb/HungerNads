# Suggested Commands

## Development
- `npm run dev` - Start Cloudflare Workers dev server (port 8787)
- `npm run typecheck` - TypeScript type checking (tsc --noEmit)
- `npm run deploy` - Deploy to Cloudflare Workers

## Testing
- `npm run test:llm` - Test LLM providers

## Database
- `npm run db:migrate` - Run D1 migrations locally

## System (macOS/Darwin)
- `git`, `ls`, `cd`, `grep`, `find` all standard
